herdr keybindings
=================

Live keymap: prefix+?    Docs: https://herdr.dev/docs/
Prefix: ctrl+b, then release and press the action key.

Start and session
-----------------
  h                       start or attach
  hr <host>               attach remotely
  prefix+q                detach; processes keep running

Direct keys
-----------
  ctrl+shift+t             new tab
  ctrl+shift+left/right    previous/next tab
  ctrl+shift+up/down       previous/next workspace
  ctrl+shift+n             new workspace
  ctrl+shift+[/]           previous/next agent
  ctrl+shift+b             toggle sidebar
  ctrl+shift+r             resize mode

  ctrl+shift+h/j/k/l       focus pane left/down/up/right
  ctrl+shift+enter         split right
  ctrl+shift+d             split down
  ctrl+shift+backspace     close pane
  ctrl+shift+z             zoom pane
  ctrl+shift+,/.           previous/next pane

  ctrl+shift+o             OpenCode in a new tab at the active directory
  F1                       edit scrollback

Prefix keys
-----------
  prefix+w                 workspace picker
  prefix+g                 goto picker
  prefix+e                 edit scrollback
  prefix+shift+p           rename pane
  prefix+shift+t           rename tab
  prefix+1..9              jump to tab
  prefix+shift+x           close tab
  prefix+shift+w           rename workspace
  prefix+shift+d           close workspace
  prefix+shift+g           new worktree
  prefix+s                 settings
  prefix+shift+r           reload config
  prefix+?                 show keymap

Notes
-----
Kitty reserves ctrl+shift+c/v and ctrl+shift+=/-. Other ctrl+shift keys pass
to herdr. New panes inherit the active directory, but each starts a new shell;
direnv therefore runs once per pane.
